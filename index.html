About SchemOrigin


In this project, ER Model Description Script (ERD) is introduced along with its compiler. ER Model Description Script (ERD) is used to represent graphical ER model to its textual equivalent. Its compiler translates given ER Model Description Script (ERD) to its equivalent Relational Database schema.

Features of Schema Generator

  1. Converts ER Model to Database Schema.
  2. It can apply 1-1, 1-M, M-N and ISA relationship to entities.
  3. Multi-Valued attributes are supported.
  4. Descriptive attributes are supported.
  5. Automatic attribute name conflict resolution.
  6. Syntax and semantic errors can be detected in ER Model Description script (ERD).
